ML Model Usage Example:
```python
import pickle
import pandas as pd
import json

# Load registry
with open('models/model_registry.json', 'r') as f:
    registry = json.load(f)

# Get module info
module_info = registry['models']['heart']

# Load model and scaler
with open(module_info['model_path'], 'rb') as f:
    model = pickle.load(f)
with open(module_info['scaler_path'], 'rb') as f:
    scaler = pickle.load(f)

# Prepare input data (must match feature_names order)
feature_names = module_info['feature_names']
X_new = pd.DataFrame([[...]], columns=feature_names)

# Scale and predict
X_scaled = scaler.transform(X_new)
prediction = model.predict(X_scaled)
probability = model.predict_proba(X_scaled)

print(f"Prediction: {prediction[0]}")
print(f"Confidence: {probability[0]}")
```
